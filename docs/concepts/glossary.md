# openpilot术语表

* **onroad**: 点火开启时openpilot的系统状态
* **offroad**: 点火关闭时openpilot的系统状态
* **route**: 一次onroad会话的记录
* **segment**: route被分割成的一分钟长的片段
* **comma connect**: 查看所有routes的网页查看器;访问[connect.comma.ai](https://connect.comma.ai)了解更多
* **panda**: 设备上实现功能安全并通过CAN直接与车辆通信的辅助处理器。详见[panda仓库](https://github.com/commaai/panda)
* **comma 3X**: comma.ai最新的运行openpilot的硬件。更多信息请访问[comma.ai/shop](https://comma.ai/shop)
