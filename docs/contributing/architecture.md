# 系统架构

openpilot是一个旨在提高驾驶安全性和便利性的驾驶代理系统。

### 感知系统

感知系统包括:

* 20Hz的前向道路摄像头视频流
* 10Hz的驾驶员监控摄像头视频流
* 100Hz的9轴IMU(提供加速度和旋转测量)
* 1Hz的GPS定位
* 100Hz的CAN传感器(提供车轮速度、加速度、转向角等数据)

### 规划系统

规划系统包括:

* 横向规划器(规划跟随车道的路径)
* 纵向规划器(规划跟随前车的速度曲线)

### 控制系统

控制系统包括:

* 横向控制器(控制方向盘执行规划的路径)
* 纵向控制器(控制油门和制动执行规划的速度)

### 驾驶员监控

驾驶员监控系统:

* 使用驾驶员监控摄像头检测驾驶员分心状态
* 在驾驶员分心时发出警告
* 如果驾驶员无响应则自动解除openpilot

### 安全系统

安全系统:

* 运行在独立处理器(panda)上
* 监控所有CAN消息是否存在安全违规
* 检测到任何安全违规时自动解除openpilot

### 日志系统

日志系统:

* 记录所有传感器数据和内部状态
* 在连接WiFi时将数据上传到comma服务器
* 支持离线分析和调试
