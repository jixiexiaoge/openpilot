## 认识「CP搭子」（CPlink）

一款专为 Comma3 设备与 CarrotPilot 打造的 Android “导航数据桥接”应用，现阶段唯一支持从高德地图读取导航数据并实时发送至设备的方案。

核心能力：
- 实时导航路线与道路名称
- 道路限速、摄像头/区间测速限速提醒
- 转弯（TBT）提示与距离
- 交通灯状态与倒计时
- GPS 位置与行驶方向

- Android 8.0+ 手机，安装高德地图与 CP搭子
- Comma3 已安装 CarrotPilot，手机与设备处于同一 WiFi
