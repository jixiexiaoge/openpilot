## 认识「CP搭子」（CPlink）

一款专为 Comma3 设备与 CarrotPilot 打造的 Android “导航数据桥接”应用，现阶段唯一支持从高德地图读取导航数据并实时发送至设备的方案。

核心能力：
- 实时导航路线与道路名称
- 道路限速、摄像头/区间测速限速提醒
- 转弯（TBT）提示与距离
- 交通灯状态与倒计时
- GPS 位置与行驶方向

工作方式（超简版）：
- 手机自动接收高德地图广播 → 转换为 CarrotMan 协议 → 通过 WiFi 发送给 Comma3
- 设备自动发现：监听 7705；数据发送：UDP 7706；频率约 200ms；JSON 负载

使用前准备：
- Android 8.0+ 手机，安装高德地图与 CP搭子
- Comma3 已安装 CarrotPilot，手机与设备处于同一 WiFi
- 授予位置与网络权限（建议开启后台定位）


了解更多与完整功能清单：参见《[CP搭子 - 应用功能详细说明](./app功能说明.md)》

### 界面与操作一览（CP搭子）

- 主页
	- 连接状态与设备信息（自动发现 / 自动重连）
	- 实时数据卡片：导航状态、道路/限速、转弯距离、交通灯等
	- 左/右速度圆环：设定巡航与当前速度展示，便于快速确认
- 高阶功能（九宫格）
	- 加速 +10 / 减速 -10
	- 左变道 / 右变道
	- 智能控速：智能控速 / 原车巡航 / 弯道减速 三模式循环
	- 设置限速：一键将道路限速设为巡航速度
	- 启用路径：OpenPilot 激活时手动确认跟随导航
